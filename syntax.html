Syntax!

Variables:

A variable is a storage unit (data type). It can hold different values which can be manipulated or changed later on. Constants, on the other hand, remain the value they are initiazed to. There are different types of variables and constants, which infer a type of value within the container. Below is a list of some variable types:

  • Number:   a numerical value specific as a number type (eg: 6).
  • String:   a text based value, it could also be a number, but is not operatable on as a number is. Strings values are surrounded by quotation marks (eg: "hello" or "6").
  • Boolean:   a binary value of true or false (true || false). - side note; || is programming speak for OR
  • Array:   a collection of different values. These would be a collection of similar types (eg: array of strings ["hello", "there"])

Conditional Statements:

A conditional statement is the expression of logic where different options are created. Based on these, different outcomes exist. There are different types of variables and constants, which infer a type of value within the container. Below is a list of some variable types:

  • IF:     if this happens, then execute code.
  • ELSE IF:   if that condition wasn't met, try these conditions and execute my code.
  • ELSE:   states that if the previous conditions weren't met, there is a default action to be taken - the conditions for "else" are set by if/else if statements.
